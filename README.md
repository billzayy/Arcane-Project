# Arcane E-commerce Project
### This is a fullstack project using: 
* HTML / CSS / JS _( Front-End )_
* NodeJS _( Back-End )_
* MySQL _( Database )_
* Docker

We have some pages for this project :

* Main Page
* Product
* Shopping Cart
* Catalogue 
* Contact
* Responsive

--------------------------------
To use this project, you need to use Docker command on your terminal to build:
```
docker-compose up -d
```
